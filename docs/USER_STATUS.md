### 🌉 코드-이론 가교 (Code-Theory Bridge)

이미 정복한 코딩 테스트의 전술은 기술 면접의 강력한 무기(CS 이론)가 됩니다.

1.  **표 편집 (Table Edit) ↔️ 연결 리스트 & 영속성 컨텍스트**
    - **코드 전술:** 인덱스 배열(`prev`, `next`)을 사용해 삭제와 복구를 $O(1)$에 수행.
    - **이론 교각:** 자료구조의 **Linked List** 구현 메커니즘이자, **JPA 영속성 컨텍스트**가 엔티티의 상태(Persistence/Detached)를 관리하며 가비지 컬렉션의 대상이 되지 않게 하는 '불멸의 인덱스' 철학과 연결됨.

2.  **주식 가격 (Stock Price) ↔️ 스택 & 메모리 구조**
    - **코드 전술:** 단조 스택(Monotonic Stack)으로 가격의 낙하 시점 역추적.
    - **이론 교각:** 자료구조 **Stack**의 LIFO 특성을 극대화한 사례. JVM의 **Stack Area**에서 메서드 프레임이 생성/소멸되는 방식과 데이터가 쌓였다가 특정 조건(return)에서 일시에 해소되는 '연쇄 폭발' 논리가 일치함.
